Hepsi Pay Java Sample Client

1) First add hepsipay-java-client-1.1.1.jar file to your local Maven Repo: 

mvn install:install-file -Dfile=hepsipay-java-client-1.1.1.jar -DgroupId=com.hepsipay.client -DartifactId=hepsipay-java-client -Dversion=1.1.1 -Dpackaging=jar

2) Extract hepsipay-client-sample.zip file

3) Edit file SampleUtil.java change SecretKey and ApiKey text in the generateSampleHepsiPaySetting() method.

Ex:

HepsiPaySetting hepsiPaySettings = new HepsiPaySetting();
hepsiPaySettings.setApiKey("ApiKey");
hepsiPaySettings.setSecretKey("SecretKey"); 
hepsiPaySettings.setTestMode(true);

4) Open the Test class and run.

Ex: SampleCardQuery.java
